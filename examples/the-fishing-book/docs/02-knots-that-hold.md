# Knots That Hold

## The Clinch Knot
1. Thread line through hook eye
2. Wrap 5–7 times around standing line
3. Pass tag end through loop above eye, then through the big loop
4. Moisten, pull tight

## The Perfection Loop (for leaders)
- Forms a non-slip loop at line end
- Excellent for loop-to-loop connections

> A bad knot loses fish. A good knot loses sleep thinking about the bad knot.
